# MOON × LANDMARK PLANNER

PLの塔・岐阜城と月（または夕日）の撮影計画を支援するWebツールです。  
インストール不要の単一HTMLファイルで動作します。

---

## 機能一覧

### 01 対象選択
- **PLの塔**（大阪府富田林市）/ **岐阜城**（岐阜県岐阜市）を切り替え
- **🌅 夕日モード**：PLの塔選択時のみ有効。月の代わりに夕日との構図を計画

### 02 日時設定
- 起動時に現在の日時（JST）を自動セット
- 日付フィールドに曜日表示（日=赤、土=青）
- **±ボタン**で時刻を微調整（±1 / ±5 / ±10 / ±15 / ±30 / ±60分）
- **NOW**ボタンで現在時刻を即セット
- 日の出・日の入り・月の出・月の入りの時刻をクリックするとその時刻をセット

### 03 焦点距離・向き・三脚高さ・標高オフセット

**焦点距離**
- スライダーまたはプリセットボタンで設定（9〜1600mm、12段階）
- スマホでは数字部分をタップするとテンキーパッドが開き直接入力可能
- 横位置・縦位置の切替対応（35mmフルサイズ基準）

**三脚高さ**
- ±1 / ±5 / ±10cmボタンで調整（範囲：10〜300cm、デフォルト150cm）
- **Rst**ボタンでデフォルト（150cm）にリセット
- 三脚高さはプレビューの仰角計算に反映（超望遠時に特に有効）

**標高オフセット**
- 地点選択後、APIで取得した地面標高に ±1 / ±3 / ±5m を加算できる
- 屋上・橋・傾斜地など、実際の目線高さを補正するための機能
- 新しい地点を選択するとオフセットは自動リセット

### 04 撮影地点の指定
3種類の方法で撮影地点を指定できます：

| 方法 | 操作 |
|---|---|
| 地図クリック | 地図上の任意の地点をクリック |
| 📍 現在地 | GPSで現在地を自動取得（ブラウザの位置情報許可が必要） |
| 📋 URLペースト | Google マップのURLをコピーしてボタンを押す |

**📋ボタン対応URLフォーマット**：
- 短縮URL（`maps.app.goo.gl/...`）
- 通常URL（`google.com/maps/@lat,lng...`）
- Place URL（`google.com/maps/place/lat,lng/...`）
- データURL（`...3d34.xxx!4d135.xxx...`）
- 度分秒形式（`34°29'58.3"N 135°32'37.9"E`）
- 括弧付き座標テキスト（`(34.123, 135.456)`）

指定後に自動で以下を取得・計算：
- 撮影地点の標高（国土地理院API）
- 対象までの水平距離・3D距離・方位角・仰角

**地図操作**：
- 撮影地点マーカー（青丸）はドラッグで移動可能
- 右上の🛰ボタンで地図／衛星画像（Esri World Imagery）を切替
- マーカータップでGoogleマップ／ストリートビューへのリンクが開く

**撮影地点の保存**：
- 💾ボタンで現在地点を名前付きで保存（セッション内）
- 保存リストをクリックすると地点を再読み込み
- 🔗ボタンで保存地点全件をURLハッシュとしてクリップボードにコピー（共有・復元可）

### 05 太陽・月情報
- 日の出 / 日の入り / 月の出 / 月の入り時刻を表示
- 月齢・月相名・月相アイコンを表示
- 各時刻をクリックすると設定時刻に反映

### 06 チャンス一覧
「📅 選択日から30日を検索」ボタンで、現在設定中の撮影地点から対象に月が**物理的に重なる日時**を自動検索します。

**判定ロジック**：
- 10分刻みで全時刻をスキャン（30日 × 144コマ）
- 月の方位角が対象方位角と **±2.5°以内**
- 月の仰角が **0°〜対象頂点の仰角** の範囲内（塔にかかっている高さ）
- 条件を満たす最初の時刻を「重なり開始」として記録

**結果表示**：
- ★★★：方位角差±1°以内 / ★★☆：±2°以内 / ★☆☆：±2.5°以内
- 月齢・方位角ずれ・対象の何%の高さに月が来るか（高XX%）を表示
- クリックで日時を一括セット → プレビューで目視確認

> チャンス一覧は絞り込み用の目安です。最終確認はプレビューで行ってください。

### プレビュー（右パネル上段）
- 指定焦点距離・時刻での空のプレビューをリアルタイム表示
- 対象シルエット（PLの塔：詳細ポリゴン輪郭 / 岐阜城：16方向3Dシルエット）と月または太陽を重ねて描画
- 月は月齢に応じた位相を描画
- 太陽は仰角に応じて色が変化（低いほど赤みが強くなる）
- 左上に現在の設定日時を常時表示
- **−1d / +1d ボタン**：プレビューの日付を前後1日ずつ変更（時刻設定は保持）

**軌道トレイル**：
- その日の月／太陽の全軌道を薄いラインで表示
- 現在時刻の前後の月／太陽位置を時刻付きの点で表示
  - 視野角から前後の表示範囲・刻みを自動計算（広角ほど広範囲、望遠ほど細かく）
  - 過去は青、未来はオレンジの点で表示
- 現在位置から進行方向への矢印を描画

**視点オフセット（⊹ボタン）**：
- プレビュー右上の⊹ボタンで8方向パッドを開く
- 方位角・仰角をそれぞれ±0.5°単位でずらし、カメラの微調整をシミュレート
- RSTボタンでリセット

**スマホ操作**（プレビューcanvas上）：
- ピンチイン／アウト：canvasをズーム（最大5倍）
- 拡大中に1本指スワイプ：canvasをパン（移動）
- ダブルタップ：ズーム・位置をリセット
- ラベル・±1dボタン・⊹ボタンはズームの影響を受けず固定

地球曲率補正実装済み（R=6371km）  
焦点距離を変えても対象の頂点角度は物理的に一定

### 方位・仰角ダイアグラム（右パネル下段）
- 撮影地点から見た対象・月（太陽）の方位を円形ダイアグラムで表示
- 仰角ダイアグラムも右側に表示
- 撮影候補エリアの扇形も重ねて表示

### 撮影候補エリア（地図上の扇形）
- 月の出・月の入り前後2時間を30分刻みで扇形表示
- 夕日モード時は日の入り前2時間を表示
- 同心円でズームレベルに応じた距離目安を表示（ズーム変更で自動更新）
- 地図左上の凡例ボックスで時間帯と扇の色を確認できる

## 使い方の流れ

1. 対象（PLの塔 or 岐阜城）を選択
2. 撮影予定日を設定
3. **06 チャンス一覧**で重なる日時を検索 → クリックで日時をセット
4. 地図の扇形エリアを参考に撮影候補地点をクリック（またはドラッグ・URLペースト）
5. プレビューで月と対象の重なりを目視確認
6. ±ボタンで時刻を微調整、三脚高さ・標高オフセットも必要に応じて調整
7. 軌道トレイルで前後の動きを確認して構図を決定
8. 気に入った地点は💾で保存、🔗でURL共有

## 計算精度

| 項目 | 精度・出典 |
|---|---|
| PLの塔座標 | 34.502231°N 135.585904°E（高さ180m、標高110m） |
| 岐阜城座標 | 35.433931°N 136.782041°E（高さ29m、標高335m） |
| 撮影地点標高 | 国土地理院標高APIによる自動取得 |
| 月位置計算 | Meeus「天文アルゴリズム」Ch.47 主要摂動項32項（誤差±0.1°以下） |
| 太陽位置計算 | Meeus「天文アルゴリズム」Ch.25（誤差±0.01°以下） |
| 大気差補正 | 標準大気（気温10℃、気圧1013hPa）の仰角依存式で自動補正 |
| 地球曲率補正 | 実装済み（R=6371km） |

## 動作環境

- モダンブラウザ（Chrome / Firefox / Safari / Edge）
- インターネット接続が必要（地図タイル・標高API・短縮URL展開）
- 位置情報・クリップボードは初回のみ許可が必要

---

## 注意事項

- プレビューはあくまで**目安**です。実際の撮影時は現地で調整してください
- 大気差は標準大気での補正値を使用しています。実際の気象条件により多少異なる場合があります
- 短縮URLの展開には外部プロキシ（corsproxy.io / allorigins.win）を経由します
- 保存地点はページをリロードすると消えます。共有URLをブックマークしてください

### 撮影提案セクション（コメントアウト中）
`index.html` 内の「07 撮影提案」セクションは現在HTMLコメントアウトで非表示。

**理由**：提案内容の指針が未定のため。推奨焦点距離・方位角・仰角などの情報は表示していたが「なぜその提案をするのか」の根拠・基準が曖昧で、ユーザーにとってわかりにくい内容だった。

**復活させるには**：
- 提案の指針・基準を決める
- `<!-- 07 撮影提案 ... -->` のコメントアウトを外す
- `suggest()` 関数の内容を指針に合わせて書き直す

# MOON × LANDMARK PLANNER

PLの塔・岐阜城と月（または夕日）の撮影計画を支援するWebツールです。  
GitHub Pagesで動作する単一HTMLファイルで、インストール不要で使用できます。

---

## 機能一覧

### 01 対象選択
- **PLの塔**（大阪府富田林市）/ **岐阜城** を切り替えて使用
- **🌅 夕日モード**：PLの塔選択時のみ有効。月の代わりに夕日との構図を計画

### 02 日時設定
- 起動時に現在の日時（JST）を自動セット
- 日付・時刻の手動入力が可能
- **±ボタン**で時刻を微調整（±1 / ±5 / ±10 / ±15 / ±30 / ±60分）
- **日の出・日の入り・月の出・月の入りをクリック**するとその時刻を即セット

### 03 焦点距離設定
- スライダーまたはプリセットボタンで設定（9〜1600mm、12段階）
- 横位置・縦位置の切替対応（センサーサイズ 35mmフルサイズ基準）

### 04 撮影地点の指定
3種類の方法で撮影地点を指定できます：

| 方法 | 操作 |
|---|---|
| 地図クリック | 地図上の任意の地点をクリック |
| 📍 現在地ボタン | GPSで現在地を自動取得（ブラウザの位置情報許可が必要） |
| 📋 URLペーストボタン | Google マップ / ストリートビューのURLをコピーしてボタンを押す |

> **📋ボタンについて**  
> 短縮URL（`maps.app.goo.gl/...`）・通常URL（`google.com/maps/...`）どちらにも対応。  
> 初回のみクリップボードへのアクセス許可が求められます。

指定後に自動で以下を取得・計算：
- 撮影地点の標高（国土地理院API）
- 対象までの距離・方位角・仰角
- 推奨焦点距離

### 05 太陽・月情報
- 日の出 / 日の入り / 月の出 / 月の入り時刻を表示
- 月齢・月相名を表示
- 各時刻をクリックすると設定時刻に反映

### 06 撮影地点からのプレビュー
- 指定した焦点距離・時刻での**空のプレビュー**をリアルタイム表示
- 対象（PLの塔 / 岐阜城）のシルエットと月（または太陽）の位置を重ねて表示
- 月は月齢に応じた位相を描画
- 太陽は仰角に応じて色が変化（低いほど赤みが強くなる）
- PLの塔には三分割の補助線あり（拡大時の位置確認用）

#### プレビューの計算仕様
- 塔の見かけの大きさは**3D距離**（水平距離＋標高差）で計算
- 三脚・カメラの目線高さとして**+1.5m**を加算
- **地球曲率補正**を実施（長距離撮影の精度向上）
- 焦点距離を変えても対象の頂点角度は一定（物理的に正確）
- 頂点が画面上部60%を超える場合は頂点位置を固定して表示

### 07 方位・仰角ダイアグラム
- 撮影地点から見た対象・月（太陽）の方位を円形ダイアグラムで表示
- 対象のELEV（仰角）を下部に表示

### 08 撮影候補エリア（扇形表示）
- 月の出・月の入り前後2時間を30分刻みで扇形表示
- 夕日モード時は日の入り前2時間を表示
- 同心円で3 / 5 / 10 / 15km の距離目安を表示
- 地図左上の**凡例ボックス**で時間帯と扇の色が確認できる

### 09 撮影提案（06パネル）
- 推奨焦点距離の自動算出
- 対象方位・仰角の評価コメント
- 撮影チャンスのタイミング案内

---

## 使い方の流れ

1. 対象（PLの塔 or 岐阜城）を選択
2. 撮影予定日を設定
3. 地図の扇形エリアを参考に撮影候補地点をクリック
4. 月の出・月の入りなどの時刻をクリックして時刻をセット
5. プレビューで月と対象の重なりを確認
6. ±ボタンで時刻を微調整して構図を決定
7. 📍または📋でフィールドの実地点を反映して再確認

## 座標・データ精度

| 項目 | 精度・出典 |
|---|---|
| PLの塔座標 | 34.502231°N 135.585904°E（国土地理院地図より） |
| 岐阜城座標 | 35.433931°N 136.782041°E（国土地理院地図より） |
| 撮影地点標高 | 国土地理院標高APIによる自動取得 |
| 月位置計算 | Meeus「天文アルゴリズム」Ch.47 主要摂動項32項（誤差±0.1°以下） |
| 太陽位置計算 | Meeus「天文アルゴリズム」Ch.25（誤差±0.01°以下） |
| 地球曲率補正 | 実装済み（R=6371km） |
| 大気差補正 | 標準大気（気温10℃、気圧1013hPa）の仰角依存式で自動補正 |

## 動作環境

- モダンブラウザ（Chrome / Firefox / Safari / Edge）
- インターネット接続が必要（地図タイル・標高API・短縮URL展開）
- 位置情報・クリップボードは初回のみ許可が必要

---

## 注意事項

- プレビューはあくまで**目安**です。実際の撮影時は現地で調整してください
- 短縮URLの展開には外部サービス（corsproxy.io）を経由します
- 大気差は標準大気（気温10℃、気圧1013hPa）での補正値を使用しています。実際の気象条件により多少異なる場合があります
